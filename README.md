
# Swiggy Sales & Restaurant Performance Analysis (SQL Server + Power BI)

## Overview
This project analyzes Swiggy food delivery data using SQL Server and Power BI to uncover sales trends,
restaurant performance, and pricing insights.

## Tools Used
- SQL Server (T-SQL)
- Power BI
- Excel

## Key Analysis
- Revenue by restaurant and city
- Monthly sales trends
- Ranking restaurants using window functions
- Views optimized for Power BI

## How to Run
1. Run `01_create_table.sql`
2. Run `02_insert_data.sql`
3. Execute analysis scripts
4. Connect Power BI to SQL Server views

## Business Insights
- A small number of restaurants contribute a large share of revenue
- Higher ratings often correlate with premium pricing
